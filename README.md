# Railway Reservation System

## Overview
The Railway Reservation System is a Java-based web application that allows users to search trains, check seat availability, and book tickets.  
The system uses a backend database to store train details, user information, and booking history.

## Features
- User registration and login
- Search trains between stations
- Check seat availability
- Book railway tickets
- Admin management for train records
- View booking history

## Technologies Used
- Java
- JDBC
- Servlets
- Oracle SQL
- HTML
- CSS
- JavaScript
- Apache Tomcat

## Setup Instructions
1. Clone the repository
2. Import the project into Eclipse
3. Configure the database connection in `application.properties`
4. Run the project on Apache Tomcat server

## Database
Oracle database is used for storing user and train data.
